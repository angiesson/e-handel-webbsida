# Stepify - An E-commerce Website for Shoes

## Project Description
This project is an e-commerce website that sells shoes for women, men, and children.  
The purpose of the project is to create a user-friendly and modern website where customers can easily browse products, view details, and make purchases.  

The website includes different categories of shoes as well as features designed to improve the user experience, such as product displays and navigation between different sections.

---

## Instructions to Run the Project

1. Download or clone the project folder to your computer

2. Open the project in a code editor (e.g. VS Code)

3. Open `homepage.html` in a web browser

---

## User Description

Users can visit the website to browse different types of shoes for women, men, and children.  
The website is designed to be easy to use and provide a smooth shopping experience.

### On the website, users can:
- Navigate between different categories (women, men, children)
- View products and their information (e.g. name, price, and image)
- Click on products to see more details
- Add and remove products from the cart
- Mark products as favorites
- Search for specific types of shoes by brand or name
